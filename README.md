# Edoc---The-Echanneling-System
A web-based E-Doc system that allows patients to schedule appointments, manage records, and intract with doctors online.

📌 About Project

E-Doc is a web-based Hospital Management System designed to digitize healthcare operations. It allows patients to search doctors, check real-time availability, and book appointments online, reducing waiting time and manual work.

🚀 Key Features

🔐 User Authentication (JWT-based)
👨‍⚕️ Doctor Portal (profile & scheduling)
📅 Real-time Appointment Booking
📊 Data Analytics Dashboard
📄 Automated PDF Generation (reports & receipts)
👥 Role-Based Access Control (Admin, Doctor, Patient)

🛠️ Tech Stack

Frontend: HTML5, CSS3, JavaScript
Backend: Python (Flask)
Database: MySQL
API: RESTful API
Security: JWT Authentication, PBKDF2 Hashing

🏗️ System Architecture

Full-stack web application
REST API based backend
Database normalized (3NF)
ACID-compliant data handling

🔒 Security Features

JWT-based authentication
Role-based access control (RBAC)
Secure password hashing (PBKDF2)

▶️ How to Run

Clone the repository
Install dependencies
pip install -r requirements.txt
Run the server
python app.py
Open browser
http://127.0.0.1:5000/

📈 Performance

Handles ~200 concurrent users
99.5% uptime during testing
